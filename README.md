# 灵巧日历

这是一个遵循 RFC5545 标准的 Android 日历应用程序，实现了完整的日历功能，包括基本的日程管理和扩展功能。

## 基本功能

### 1. 日历视图展示
- **月视图**：显示完整的月份日历网格，包含当前月份以及前后月份的日期
- **周视图**：以列表形式展示一周的日程安排
- **日视图**：详细展示特定日期的日程安排

### 2. 日程管理
- **添加事件**：支持设置标题、描述、地点、时间、类型等属性，可以设置重复规则和添加添加日程提醒
- **编辑事件**：可以修改已有事件的各项属性
- **查看事件**：支持查看某天的所有日程列表以及某个日程详情
- **删除事件**：移除不需要的日程

### 3. 日程提醒功能
- **提醒设置**：支持为事件设置多个提醒时间
- **通知显示**：在设定时间通过通知提醒用户
- **重复事件**：支持按日、周、月、年重复的事件

## 扩展功能

### 1. 日历事件导入导出
- **ICS格式支持**：完全兼容 RFC5545 标准的 ICS 文件格式
- **导入功能**：可以从外部 ICS 文件导入事件
- **导出功能**：可以将事件导出为标准 ICS 文件

### 2. 网络订阅功能
- **订阅管理**：支持添加、编辑和删除网络日历订阅
- **自动同步**：定期自动同步订阅的网络日历数据
- **后台服务**：通过前台服务持续运行同步任务
- **开机自启**：设备重启后自动恢复订阅同步服务

### 3. 农历支持
- **农历转换**：精确的公历到农历日期转换
- **农历节日**：识别并显示传统农历节日

## 技术特点

### 数据存储
- 使用 Room 数据库进行本地数据持久化
- 支持事件、提醒和订阅三种数据实体

### 系统集成
- 通知系统集成，提供及时提醒
- 开机广播接收器，确保服务持续运行
- 权限管理系统，合理申请所需权限

### 用户体验
- Material Design 设计风格
- 流畅的视图切换体验
- 直观的操作界面

## 使用说明

1. **日历视图查看**：通过底部 Tab 在月、周、日视图间切换
2. **日程管理**：点击底部"添加事件"按钮，填写事件信息并保存，之后点击带有蓝色圆点的日期可查看日程详情，进行修改和删除
3. **设置提醒**：在添加或编辑事件时设置提醒时间
4. **导入导出**：使用导入/导出功能与其他日历应用交换数据
5. **网络订阅**：添加网络日历订阅以同步共享日程

## 技术实现细节

### RFC5545 标准支持
- 完整实现 VEVENT 组件
- 支持 SUMMARY、DESCRIPTION、LOCATION 等标准属性
- 支持 DTSTART、DTEND 时间属性
- 支持 RRULE 重复规则

### 农历算法
采用精确的天文算法计算农历日期，支持 1900-2100 年的农历数据。

### 同步机制
通过 Android 的前台服务实现稳定的后台同步，确保订阅数据实时更新。
